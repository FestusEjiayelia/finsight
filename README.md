# FinSight — Personal Finance Analyzer

> 3MTT NextGen Knowledge Showcase | Financial Inclusion Pillar

## Live Demo
https://festusejiayelia.github.io/finsight

## What it does
FinSight turns your Nigerian bank statement into an instant financial dashboard. Upload your OPay, Kuda, or GTBank CSV statement and get:

- Monthly income vs spending breakdown
- Automatic transaction categorization
- Financial health score
- Savings goal predictor with 3 projection models
- Financial literacy tips personalized to your data

## The Problem
Over 40 million Nigerians use mobile wallets but have no visibility into their financial behaviour. FinSight changes that — no signup, no server, works entirely in your browser.

## Built With
- HTML, CSS, JavaScript
- Chart.js for visualizations
- Claude AI (Anthropic) — used for data analysis, categorization logic, and dashboard development

## AI Disclosure
This MVP was built using Claude AI as a development assistant. Claude was used to parse and analyze the OPay transaction dataset, build the categorization engine, design the savings projection models, and develop the dashboard interface.
